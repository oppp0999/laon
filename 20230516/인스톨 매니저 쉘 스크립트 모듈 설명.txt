ODBC -> 오픈형 데이터베이스에 접근하기 위한 표준 구격
JDBC -> 자바 데이터베이스

yarn -> node.js 자바스크립트 등 패키지 관리자

libarchive -> 스트리밍 아카이브 작업을 위한 오픈 라이브러리

dbeaver -> sql 클라이언트이자 데이터베이스 관리 도구

mqtt -> 머신 대 머딘 통신에 사용되는 표준 기반 메시징 프로토콜 또는 규칙 세트 (tcp/ip 프로토콜에서 동작)

tensorflow -> 엔드 투 엔드 오픈소스 머신러닝 플랫폼
E2EE -> 엔드 투 엔드 제3자가 엔드포인트 간에 전송된 데이터에 액세스하지 못하도록 차단하는 보안 통신 프로세스

FFmpeg -> 디지털 음성 스트림, 영상 스트림에 대해서 다양한 종류의 형태로 기록하고 변환하는 컴퓨터 프로그램

Darknet -> 신경망 프레임워크 로서 dnn들을 학습시키고 실행 시킬 수 있는 툴
YOLO -> 이미지에서 객체를 탐지하는 모델

TAS ->별도의 외부 솔루션 없이 직접 디스크 장치를 관리
